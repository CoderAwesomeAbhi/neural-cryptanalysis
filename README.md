# Neural Cryptanalysis: Period Growth and Neural Resistance in p-adic Dynamical Systems

A computational and theoretical study of neural network limitations on periodic algebraic sequences generated by piecewise affine maps over Z/p^k Z.

## Overview

This project investigates fundamental limits of neural networks when learning periodic sequences with algebraic structure. We prove super-linear period growth theorems, quantify neural resistance thresholds, and establish information-theoretic barriers.

## Key Results

### 1. Super-Linear Period Growth (Theorem 3.1)
- **Proved:** T(p^(k+1)) > p · T(p^k) for Hensel-satisfied configurations
- **Validated:** 168 primes with mean growth ratio 26.94× (max 79.42×)
- **Significance:** Novel connection between Hensel's Lemma and dynamical systems

### 2. Neural Resistance Threshold (Observation 3.5)
- **Quantified:** T/L_in ≤ 21 → 100% accuracy; T/L_in ≥ 181 → <17% accuracy
- **Architecture-independent:** MLP, LSTM, Transformer, SSM all exhibit same threshold
- **Predictive:** Can calculate memory requirements for sequence learning

### 3. Information-Theoretic Barrier (Section 8)
- **Tested:** p-adic positional encoding, context scaling, architectural modifications
- **Result:** No significant improvement (barrier is fundamental, not architectural)
- **Implication:** Establishes limits of gradient-based learning on algebraic sequences

### 4. LC-NC Separation (Theorem 12.1)
- **Proved:** Linear Complexity and Neural Complexity are independent measures
- **Constructed:** Explicit sequences with high LC + low NC and vice versa
- **Significance:** LC is not a valid proxy for neural hardness

## Key Contributions

1. **Mathematical:** First systematic study of period growth in piecewise affine systems over Z/p^k Z with Hensel lifting
2. **Empirical:** Quantification of neural resistance threshold with predictive power
3. **Theoretical:** Establishment of information-theoretic barriers (not architectural)
4. **Methodological:** Computational verification framework for all major claims
